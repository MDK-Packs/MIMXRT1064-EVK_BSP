The 'Blinky' project is a simple CMSIS RTOS2 based example.
Compliant to Cortex Microcontroller Software Interface Standard (CMSIS).

Example functionality:

 - Virtual LED vioLED0 is blinking with a predefined period
 - Virtual Button vioBUTTON0 is used to switch Virtual LED blinking period

 - Target options:
   - Compiler:                  ARM Compiler optimization Level 1
   - Compiler:Event Recorder:   Enabled
   - CMSIS:RTOS2:Keil RTX5:     Source

 - Hardware Mapping:
            Virtual I/O      Board
   LED      vioLED0          USER_LED
   Button   vioBUTTON0       USER_BUTTON
